## 🎊 前言

本项目是基于SpringBoot + Vue + Antd Vue组件库构建的模板后台管理系统，菜单页面控制、用户认证授权功能、系统控制等功能已经完成，
大家可以在本系统的基础上继续开发，做出更强的系统。所有前端后台代码封装过后十分精简易上手，出错概率低。
